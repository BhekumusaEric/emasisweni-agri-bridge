# Content Management Guide

## Content Strategy

### Brand Voice
- **Professional**: Authoritative yet approachable
- **Community-focused**: Emphasizing local impact
- **Sustainable**: Environmental consciousness
- **Empowering**: Positive transformation messaging

### Key Messages
1. **Sustainable Agriculture**: Eco-friendly farming practices
2. **Community Empowerment**: Local ownership and development
3. **Waste-to-Wealth**: Circular economy solutions
4. **Market Access**: Connecting farmers to opportunities
5. **Skills Development**: Training and mentorship programs

- **Metrics Dashboard**: Key performance indicators
- **Testimonials**: Authentic community voices

### News Page (news.html)
- **Recent Updates**: Latest organizational news
- **Event Announcements**: Upcoming activities

### Contact Page (contact.html)
- **Contact Information**: Phone, email, address

## SEO Content Guidelines

### Title Tags (50-60 characters)
```
Homepage: "Emasisweni Agri-Bridge - Sustainable Agriculture KwaZulu-Natal"
About: "About Us - Agricultural Community Development | Emasisweni"
Programs: "Agriculture Training Programs | Emasisweni Agri-Bridge"
```

### Meta Descriptions (150-160 characters)
- Include target keywords naturally
- Mention location (KwaZulu-Natal)
- Include call-to-action
- Highlight unique value proposition

### Content Keywords
- **Primary**: sustainable agriculture, KwaZulu-Natal farming
- **Secondary**: smallholder farmers, agricultural training
- **Long-tail**: waste management agriculture South Africa
- **Local**: Impendle farming, Pietermaritzburg agriculture

### Content Structure
- **H1**: One per page, include primary keyword
- **H2-H6**: Logical hierarchy, include related keywords
- **Paragraphs**: 2-3 sentences, scannable content
- **Lists**: Break up long content blocks
- **Images**: Alt text with descriptive keywords

## Image Management

### Image Requirements
- **Format**: JPG for photos, PNG for graphics
- **Size**: Maximum 1MB per image
- **Dimensions**: Minimum 1200px width for hero images
- **Quality**: High resolution, professional appearance

### Image Optimization
```bash
# Compress images before upload
imagemin input.jpg --out-dir=output --plugin=imagemin-mozjpeg
```

### Alt Text Guidelines
- Descriptive and specific
- Include relevant keywords naturally
- Maximum 125 characters
- Avoid "image of" or "picture of"

### Image Naming Convention
```
hero-vegetable-farming-action.jpg
leadership-lindani-duma-chairperson.jpg
program-waste-management-training.jpg
```
